# 旅游行程规划技能

## 功能概述
本技能专门用于创建专业的旅游行程规划，包括：
- 多城市旅游路线设计
- 景点推荐和游玩时间安排
- 当地美食推荐
- 交通路线规划
- 住宿建议
- 实用旅游信息

## 调用时机
- 用户请求旅游行程规划
- 需要设计多城市旅游路线
- 提供景点和美食推荐
- 需要交通和住宿建议

## 输出格式
规划结果以HTML页面形式呈现，包含：
1. 行程标题（日期、天气、目的地）
2. 每日行程概览
3. 详细时间表
4. 交通信息
5. 食宿推荐
6. 实用信息
7. 高德地图集成

## 设计规范
- 网格布局，分区清晰
- 打印友好设计
- 使用图标区分活动类型
- 简约现代风格
- 景区图片卡片展示
- 高德地图路线集成
